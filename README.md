Enables 60FPS on YouTube on TV (Leanback) interface for 720p+ resolutions, with added user agent spoofing through a PS4 Pro Cobalt user agent, D-Input support, and event blocking for background play. Also, right click is now set as back to mimic Android TVs that treat right click as back. For D-Input, it mimics that of PlayStation's YouTube on TV app. X for enter, O for back, R1 for next, L1 for previous, L2 for rewind, R2 for fast forward, Options for pause, Triangle for search, Square for backspace.  

**Note: To allow resolutions like 4K, specifically decrease zoom size of the page through your browser and then reload. Keep trying until it works. Or, turn on stats for nerds and keep decreasing until viewport reports a resolution that matches or exceeds your desired resolution, and then reload.**  

**Note: This code has been made completely with Google AI Studio's app creating features. I just asked it to create it, I keep asking it to change stuff with the code, and I retest the code to make sure it works good enough.**  

**To import this extension, turn on dev mode on extensions, load unpacked, and select the root directory of the extension (which is the one that directly contains the files, which is the folder named "YT on TV 60fps Enabler, UA, D-Input Support, and Background Play"), and try to test it for yourself if it works!**
